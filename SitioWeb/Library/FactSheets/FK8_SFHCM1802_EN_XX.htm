SAP ERP: Cost Center Integration to Employee Central Payroll

Technical details

Configuration guide

This building block contains the configuration settings for the Cost Center Integration between the SAP ERP Finance system and SAP SuccessFactors Employee Central Payroll system. To ensure that the cost centers are in sync with the On-Premise SAP ERP Finance system, this building block provides a delta distribution setup based on Application Link Enabling (ALE) and IDoc interface technology.

Function list

The following functions are provided to support your Cost Center delta distribution:

In your SAP ERP Finance system:

•Defining Logical System

•Assigning Logical System to Client

•Checking Cross-System Company Code

•Checking Cross-System Business Area

•Maintaining Distribution Model

•Activating Change Pointer

•Maintaining Port

•Creating Partner Profile

•Running Initial Distribution

•Defining Job for Delta Distribution
