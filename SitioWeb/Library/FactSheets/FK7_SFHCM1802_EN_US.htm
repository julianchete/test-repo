Employee Central: Master Data Integration with Employee Central Payroll

Technical details

Configuration guide

This building block contains the configuration settings for the Integration between your SAP SuccessFactors Employee Central Master Data and SAP SuccessFactors Employee Central Payroll. This configuration guide is only required if you plan to use SAP SuccessFactors Employee Central Payroll.

Function list

The following functions are provided to set up the Integration between Master Data and Payroll:

In your SAP SuccessFactors Employee Central system:

•Refining Succession Data Model

•Refining MDF Payment Information

•Refining Corporate Data Model

•Refining MDF Objects

•Defining/Uploading Picklists

•Checking Pay Scale Setup

•Setting Up Custom Field: Work Schedules

•Configuring People Profile

•Enhancing Role-Based Permissions

•Setting Up Payroll Configuration

•Enabling Earliest Retro Check (for Payroll)

In your SAP SuccessFactors Employee Central Payroll system:

•Defining Key Mapping of Organizational Terms

•Defining Mapping Code Value Lists

•Configuring Wage Type Processing

•Replicating Employee Key Mapping Table

In your Time Type Replication system:

•Configuring Employee Central Payroll for Time Data Integration

•Time Types: Defining Mapping Code Value Lists

In your Time Sheet Replication system:

•Configuring Employee Central Payroll for Time Sheet Integration

•Defining Mapping Code Value Lists
