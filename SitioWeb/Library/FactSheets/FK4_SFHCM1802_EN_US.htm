Employee Central: Setup Time Off

This building block contains the configuration settings for your SAP SuccessFactors Employee Central system for Time Off. Since Time Off is both country specific and employee specific, you need to specify Time-Off settings for individual employees.

Function list

The following functions are provided to set up your Time Off:

•Checking Holiday Calendar

•Checking Account Management

•Checking Time Types (Absence) and Time Profiles

•Managing Time-Off Calendars

•Enabling Accruals Time Account Process Simulator

•Enable Period-End Processing
