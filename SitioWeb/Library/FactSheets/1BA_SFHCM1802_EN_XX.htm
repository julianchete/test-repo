Employee Central: Base Setup of Data Model and Admin Center

This building block describes the base settings of data model and admin center delivered as part of this SAP Best Practices solution in case the direct usage of the preconfigured tenant (via Upgrade Center) cannot be achieved. The description supplements the existing configuration documentation and provides additional information where required.
